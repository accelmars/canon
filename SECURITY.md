# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.0.x (pre-release) | Yes — report all issues |

## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Email: security@accelmars.com

**Please include:**
- canon version affected
- Reproduction steps
- Impact assessment
- Any suggested fix (optional)

We acknowledge reports within 48 hours and aim to resolve critical issues within 14 days.

## Scope

**In scope:** Vulnerabilities in canon's own code in this repository.

**Out of scope:** Third-party dependencies (report to their maintainers), infrastructure, or social engineering attacks.
